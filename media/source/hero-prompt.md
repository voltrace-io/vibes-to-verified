# GPT Image 2 Hero Prompt

**Model:** GPT Image 2 High

**Mode:** text-to-image

**Requested aspect:** landscape

## Prompt

Create a premium editorial cover illustration for an article about multi-agent AI verification. Wide 16:9 composition, dark graphite near-black background, high-end technical magazine aesthetic, restrained and intelligent, not cyberpunk. Visual metaphor: on the LEFT, a dense field of many identical small agent nodes all following nearly identical pale paths and converging into one polished but subtly fractured central conclusion, showing correlated consensus and a shared blind spot. On the RIGHT, a smaller set of distinct agent nodes follows genuinely different routes through clear adversarial checkpoints; weak claims visibly fracture and fall away in muted red, while only a few evidence-bearing paths survive in electric mint green. A thin architectural grid and precise schematic lines connect the composition. Deep negative space across the upper-left and center for later headline typography. Color palette: charcoal black, warm ivory, muted steel blue, electric mint green, sparing coral red. Sophisticated, crisp, dimensional but minimal, editorial systems diagram meets abstract data art. No humans, no robots, no brains, no logos, no brand marks, no words, no letters, no numbers, no UI screenshots, no random code, no glowing neon city, no holograms. Ensure the story reads left-to-right and remains legible when cropped for an X article cover.

## Deterministic Composition

The selected source image is stored as `hero-base-gpt-image-2.png`. The tracked file is a pixel-identical, metadata-sanitized export; provider C2PA/JUMBF identifiers and timestamps are not included in the public candidate. Exact headline typography and the final 1600 × 900 composition are produced by:

```bash
python media/compose_hero.py
```

The final export is:

```text
media/exports/article-cover.png
```
