# Manim Production Plan

## Objective

Create a native vertical animation that explains the core Vibes to Verified argument without narration:

```text
many agents
→ correlated consensus
→ independent approaches
→ adversarial refutation
→ evidence maturity
```

The video must work muted on X and remain readable on a phone.

## Output Contract

- Aspect ratio: 9:16
- Production resolution: 1080 × 1920
- Frame rate: 30 fps
- Target duration: 26–34 seconds
- Background: `#090B0D`
- Exact typography: Segoe UI and Consolas
- No generated text or simulated software UI
- No repository URL until the public destination exists

## Beat 1 — Shared Blind Spot

**Purpose:** show that agent volume is not independence.

- Reveal the sentence: `64 agents can still share one blind spot.`
- Animate an 8 × 8 grid of identical agent nodes.
- Collapse all paths into one fractured conclusion.
- Replace the headline with: `agreement ≠ verification`

## Beat 2 — Independent Approaches

**Purpose:** establish portfolio diversity.

- Show one exact claim.
- Branch into five roles:
  - builder
  - invariant
  - adversary
  - operator
  - evidence
- Preserve distinct route colors.
- Converge only into an explicit approach registry.

## Beat 3 — Refutation

**Purpose:** make falsification visible.

- Convert approaches into atomic claim cards.
- Pass cards through an independent refutation gate.
- Break two weak claims in red.
- Allow two surviving claims to continue in mint green.
- Show: `reward agents for breaking the answer.`

## Beat 4 — V2V Scale

**Purpose:** teach the reusable model.

Reveal sequentially:

- V0 VIBES
- V1 GROUNDED
- V2 TESTED
- V3 ADVERSARIAL
- V4 OPERATIONAL

End with: `confidence is not evidence.`

## Beat 5 — Identity

- Fade to clean dark field.
- Reveal `VIBES TO VERIFIED`.
- Reveal `make your agents prove it.`
- Reveal `keep the speed. add the proof.`

## Acceptance Criteria

- All copy is readable at 1080 × 1920 on a phone-sized preview.
- No object crosses the title safe margins.
- The first frame communicates the topic within three seconds.
- The sequence works with audio muted.
- Failed claims and surviving claims are visually distinct.
- The V0–V4 order and labels are exact.
- Final video has nonzero duration and valid H.264/AAC-compatible MP4 container output.
- Representative frames pass visual inspection.
